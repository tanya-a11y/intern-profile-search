# Intern Profile Search Platform

A full-stack web application designed to manage and search intern profiles efficiently.  
This project allows users to store, view, and search intern details such as skills, education, projects, and social links through a clean API-driven architecture.

---

##  Features

-  Create and manage intern profiles
-  Search interns by skills and other attributes
-  Structured backend using FastAPI
-  Database integration with SQLAlchemy
-  RESTful APIs for frontend integration
-  Fast and scalable backend architecture
